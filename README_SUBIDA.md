# Instrucciones para subir el repositorio

1. Asegúrate de tener tu usuario y correo configurados:
   git config --global user.name "Tu Nombre"
   git config --global user.email "tu@email.com"

2. Inicializa el repositorio (si no existe):
   git init

3. Agrega los archivos:
   git add .

4. Haz un commit:
   git commit -m "Proyecto Breaking Bad CRUD listo"

5. Crea el repo en GitHub y sigue las instrucciones para agregar el remote, por ejemplo:
   git remote add origin https://github.com/tuusuario/tu-repo.git
   git branch -M main
   git push -u origin main

**Nota:**
- No subas archivos de la carpeta `src/data` ni imágenes personales, solo deja el README.txt en `img/`.
- Puedes agregar imágenes de ejemplo en la carpeta `img/` si lo deseas.
